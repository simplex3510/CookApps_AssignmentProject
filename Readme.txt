구현 요구사항 요약 정보

1. 스테이지 구현
1.1. JSON 파일을 serialize하여 Chapter와 Stage 정보 구성 기능 구현
1.2. deserialize를 통한 참조로 동적으로 스테이지 Load 기능 구현
1.3. 기간적 한계로 적극적으로 활용하지는 못함

2. 캐릭터 및 몹
2.1. FSM을 활용하여 행동양식 지정 및 실행

3. 전투 플로우
3.1. 충분히 만들 수 있으나 기간적 한계 및 기반 시스템의 구현 미달로 인하여 구현하지 못함
